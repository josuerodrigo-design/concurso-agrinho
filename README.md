# Projeto Agrinho - Conexões que Transformam

Este repositório foi desenvolvido para o trabalho avaliativo da disciplina de **Educação Digital**.

## 🌾 Sobre o Projeto
O objetivo deste site é apresentar a integração entre o campo e a cidade, destacando a importância da tecnologia, sustentabilidade e inovação no agronegócio e na nossa rotina urbana.

## 💻 Tecnologias Utilizadas
* **HTML5** (Estruturação do conteúdo)
* **CSS3** (Estilização e design visual)
* **JavaScript** (Interatividade e funções dinâmicas)
